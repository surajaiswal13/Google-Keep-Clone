# Google-Keep-Clone
A Clone of Google Keep made using Html, Css, JavaScript, ReactJS, Material UI, etc

## Steps for getting started

### 1. Install dependencies

```
npm install
```

### 2. Start the app

```
npm start
```

### 3. Open the localhost:3000 to view the app

```
http://localhost:3000/
```
